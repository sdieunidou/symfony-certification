# Roadmap et Versions Symfony

*Ce fichier détaille le calendrier des sorties. Voir aussi [Release Management](./release-management.md).*

## Calendrier
Symfony publie une nouvelle version mineure tous les 6 mois.
*   **Mai** : version x.1, x.3
*   **Novembre** : version x.0, x.2, x.4 (LTS)

## Versions Actuelles (Contexte Certif Symfony 7)
*   **Symfony 7.0** : Sortie Nov 2023. Support standard (8 mois). C'est la base de la certification.
*   **Symfony 7.1** : Mai 2024. (Hors périmètre examen).

## Cycle de Vie d'une Version
1.  **Dev** : Développement de la version.
2.  **Stabilisation (Beta/RC)** : 2 mois avant la sortie. Feature freeze.
3.  **Stable** : Sortie officielle.
4.  **Maintenance (Bugs)** : 8 mois (Standard) ou 3 ans (LTS).
5.  **Sécurité** : +8 mois (Standard) ou +1 an (LTS).
6.  **End of Life (EOL)** : Plus aucune mise à jour.

## Points de vigilance (Certification)
*   Savoir identifier la version LTS courante et future.
*   Comprendre que Symfony 7.0 est strictement identique à 6.4 en termes de fonctionnalités, moins les dépréciations. C'est le meilleur moyen de comprendre ce qui a été retiré.
