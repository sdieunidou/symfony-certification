# Licence

## Concept clé
Symfony est un Logiciel Libre (Open Source). La licence définit les droits d'utilisation, de modification et de distribution.

## Application dans Symfony 7.0
Symfony est distribué sous la licence **MIT**.

### Caractéristiques de la licence MIT :
*   **Permissive** : Très peu de restrictions.
*   **Utilisation commerciale** : Autorisée. Vous pouvez vendre une application construite avec Symfony.
*   **Modification** : Autorisée.
*   **Distribution** : Autorisée (code source ou binaire).
*   **Sous-licence** : Autorisée.
*   **Condition** : La notice de copyright et de permission doit être incluse dans toutes les copies du logiciel.
*   **Garantie** : Aucune garantie (AS IS).

## Points de vigilance (Certification)
*   Il n'y a pas de licence "Dual" (GPL + Commercial) comme pour certains produits (ex: MySQL, Qt). C'est MIT pour tout le monde.
*   La documentation est sous licence **Creative Commons BY-SA 3.0**.
*   Vous n'êtes **pas** obligé de rendre votre code Open Source si vous utilisez Symfony (contrairement à la GPL).
